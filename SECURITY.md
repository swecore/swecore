# Security Policy

Please report security issues to
[security@swecore.dev](mailto:security@swecore.dev).

Do not file sensitive vulnerabilities as public GitHub issues.

## Current scope

SWECore is in early access. This public repository is a product showcase and
documentation surface, not an enterprise or security certification.

## Reporting rules

- No public bug bounty is available yet.
- No guaranteed response SLA is provided.
- Do not perform destructive testing.
- Do not access, modify, delete, or exfiltrate data.
- Do not submit secrets, customer data, source code, private repository URLs,
  production logs, tokens, or confidential information.

## Non-claims

The current early-access security posture is not an enterprise certification,
security certification, compliance certification, or production readiness claim.
