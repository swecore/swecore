# Roadmap Boundaries

The following items are roadmap-only and are not available in early access:

- hosted dashboard;
- SSO/RBAC;
- SIEM export;
- customer-managed key;
- air-gapped install;
- compliance pack;
- enterprise approval workflow;
- enterprise control plane;
- full runtime parity.

These are not available in early access. They should be marked available only
when separately shipped and proven.
